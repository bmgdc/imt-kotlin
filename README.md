# IMT Kotlin

A Kotlin port of the Portuguese property transfer tax (IMT) calculator, built end to end
with Claude Code as a documented exercise in AI-agentic development.

This repository was initialized with a `CLAUDE.md` playbook, the reference JS engine
(`oracle/`), and the original tool's frontend for reference (`frontend-reference/`).
Everything else — the Kotlin domain logic, the API, the frontend, the tests — was built
from the prompts recorded verbatim in `prompts/`, one Claude Code session at a time. See
`DEVLOG.md` for the session-by-session log.

This README will be replaced with the full writeup once the port is complete.
